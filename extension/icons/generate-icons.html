Extension Icon Generator

Generate PNG icons for the Chrome extension

Note: This tool generates simple colored square icons. For production use, consider designing a proper icon with a logo or symbol.

16x16 px

48x48 px

128x128 px

Usage Instructions:

  1. Click the "Download All" button to download all three icon sizes
  2. Save the downloaded files as:
    • icon16.png
    • icon48.png
    • icon128.png
  3. Place them in the extension/icons/ folder
  4. Reload the extension in Chrome

Alternative: You can also use the SVG file (icon.svg) and convert it to PNG using online tools or image editing software.
