# Spooky Vocab - Chrome 商店信息

## 扩展名称
Spooky Vocab - 智能英语单词助手

## 简短描述 (132字符以内)
智能识别网页生词，显示翻译，支持间隔重复学习，帮你高效记忆英语单词。

## 详细描述

🐱👻 Spooky Vocab - 你的智能英语学习伙伴

在浏览英文网页时，Spooky Vocab 会智能识别出对你来说可能是生词的单词，并高亮显示。悬停即可查看翻译和详细释义，让你在阅读中自然习得新词汇。

### ✨ 核心功能

📖 **智能生词识别**
- 基于 CEFR 欧洲语言标准（A1-C2）自动判断词汇难度
- 根据你的英语水平，只高亮你可能不认识的单词
- 支持自定义词汇等级

🔍 **即时翻译**
- 悬停单词即可查看中文翻译
- 显示音标、词性、例句
- 无需离开当前页面

🧠 **间隔重复学习 (FSRS)**
- 采用先进的 FSRS 算法优化复习时间
- 在最佳时机提醒你复习
- 科学记忆，事半功倍

📊 **学习统计**
- 追踪你的学习进度
- 查看已掌握和正在学习的词汇
- 个性化学习数据

### 🎯 适合人群

- 英语学习者（初级到高级）
- 需要阅读大量英文资料的学生/职场人士
- 想要在日常浏览中积累词汇的用户
- 准备英语考试（四六级、雅思、托福等）的考生

### 🔒 隐私保护

- 不收集个人浏览历史
- 词汇数据安全存储
- 可随时导出或删除你的数据

### 💡 使用方法

1. 安装扩展后，点击工具栏图标登录
2. 设置你的英语水平（A1-C2）
3. 开始浏览英文网页，生词会自动高亮
4. 悬停查看翻译，点击标记"认识"或"不认识"
5. 定期使用复习功能巩固记忆

# Chrome Web Store 权限说明

## 需要请求 storage 的理由
本扩展使用 storage 权限来保存用户的学习数据，包括：已学习的单词、学习进度、用户偏好设置（如英语水平 CEFR 等级）、登录凭证（JWT token）。这些数据存储在本地，确保用户的学习记录在浏览器会话之间持久保存，无需重复登录或重置进度。

## 需要请求 activeTab 的理由
本扩展需要 activeTab 权限来读取当前页面的文本内容，以便识别并高亮显示用户可能不认识的英语单词。当用户悬停在高亮单词上时，扩展会显示翻译和释义。此权限仅在用户主动浏览页面时使用，不会在后台访问任何页面。

## 需要请求主机权限的理由
本扩展请求 https://kuroneko.chat/* 的主机权限，用于：
1. 与后端 API 服务器通信，获取单词翻译和释义
2. 同步用户的学习进度到云端
3. 加载用户登录认证所需的 SDK
此权限仅限于与扩展自有服务器通信，不会访问任何第三方网站。

## 需要请求所有网站权限的理由（content_scripts）
本扩展的核心功能是在用户浏览任意英文网页时，智能识别并高亮显示生词。为了实现这一功能，扩展需要读取页面文本内容来分析词汇难度。扩展不会收集或上传页面内容，仅在本地进行词汇分析。所有网络请求仅发送到扩展自有服务器（your-domain.com）获取翻译。

## 单一用途说明
Spooky Vocab 是一款英语单词学习助手。它的单一用途是：帮助用户在浏览英文网页时学习生词。扩展会智能识别页面中用户可能不认识的单词，高亮显示并提供即时翻译，同时运用间隔重复算法帮助用户科学记忆词汇。
